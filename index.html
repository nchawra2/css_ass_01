What is CSS?

  1. CSS stands for cascading style sheets.
  2. CSS describe how HTML element are to be displayed on screen, paper or in other media.
  3. Css is not necessary CSS saves a lot of time. It can control the layout of multiple webpage all at once
  4. External stylesheet are stored in CSS files

CSS can be added to HTML document in 3 ways:

  • Inline - by using style attribute inside HTML elements
  • Internal - by using style elememnt in head section
  • External - by using link element to link to an external CSS file. However in this tutorial we will use internal and inline styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and itâ€™s used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
